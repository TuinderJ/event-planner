# Party Planner App

## Overview
When you're hosting an event and want to track the people that are coming and if they're bringing anything for the event, this app can help you do that.

It will track:
- Attendance
    - People invited but haven't responded
    - People saying they will be attending
    - People saying the might attend
    - People saying they will not be attending
- What food/drink items attendees are planning on bringing (if you want them to)

## Host View
As a host, you will be able to edit the details of the event such as:
- Event name
- Event date
- Event time
- People that are invited
- Status of any person that's invited (including removing someone that's invited)
- Food items
    - If food items will be brought by guests
    - Add/remove suggestions
    - Add/remove required food/drink items
    - Assign guests to a food/drink item

## Guest View
As a guest, you will be able to view the details of the event such as:
- Event name and host
- Date and time of the event
- The status of all guests

As a guest, you will be able to edit information relative to you such as:
- Your name
- Your phone number
- Your attendance status
- What food items you are bringing if the host has requested it
